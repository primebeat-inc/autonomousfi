# Support

## Where to get help

- **Bugs and reproducible errors**: open a GitHub issue at the project repository. Include version, environment, and reproduction steps.
- **Design questions, architecture debates, RFC discussion**: use GitHub Discussions. These are not bugs; they are conversations.
- **Security vulnerabilities**: do NOT open public issues. Follow `SECURITY.md` and email tatsunari.shibuya at prime-beat.com.
- **Real-time chat**: AUTON Program Discord (link will be published when the channel is opened to the public, target 2026 Q3). Until then, asynchronous channels above are the only supported entry points.

## What support level to expect

This project is alpha. Phase 1 SDK is a research artifact. There is no SLA, no on-call rotation, and no guaranteed response window for general support requests. Best-effort responses only.

- Issue triage: typically within 7 days
- Pull request review: typically within 14 days, longer during audit prep weeks
- Discord questions (post launch): no SLA, community-driven

Formal SLA support arrives only with the Phase 4 SaaS launch (target 2027). Until then, treat this repository as you would any pre-1.0 research codebase.
